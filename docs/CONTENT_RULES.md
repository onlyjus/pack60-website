# Public Content Rules

This is a public website for a youth organization. Treat the repository as public.

## Public-safe checklist

- Uses general pack information only
- Avoids youth full names
- Avoids parent personal contact information
- Avoids private addresses and meeting links
- Avoids live location details
- Uses approved public-safe photos only
- Uses placeholders when facts are not approved
- Keeps private pack operations outside the site

## Forbidden public content

Do not publish:

- Youth rosters
- Youth last names
- Youth contact information
- Parent contact information unless explicitly approved
- Medical forms
- Permission slips containing personal data
- Payment records
- Individual advancement records
- Private addresses
- Private home meeting locations
- Exact travel itineraries
- Live location details during outings
- Zoom, Google Meet, or other private meeting links
- Behavior or discipline notes
- Sensitive family information
- Background check information
- Private Scoutbook data

## Youth name policy

Do not publish youth full names. Avoid youth last names in page text, captions, alt text, filenames, documents, and metadata.

## Calendar policy

The public calendar should include only public-safe event information. Registered families receive full details privately.

## Contact information policy

Use generic aliases such as `info@pack60.org`, `membership@pack60.org`, and `webmaster@pack60.org`. Do not publish personal phone numbers or private email addresses unless explicitly approved.

## Private data policy

Private pack operations belong in approved private systems, not this public repository.

## Repository safety rules

Assume every committed file may become public. Review images, documents, metadata, Markdown, and filenames before publishing.
